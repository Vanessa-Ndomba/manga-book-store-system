# Contributing

Thank you for considering contributing to the Manga Book Store System!

## Setup Instructions
1. Prerequisites:
   - Install [Python 3.10+](https://www.python.org/downloads/)
   - Install [PyCharm](https://www.jetbrains.com/pycharm/)
   - Clone the repository:
     ```bash
     git clone https://github.com/Vanessa-Ndomba/manga-book-store-system.git
     cd manga-book-store-system
     ```
   - Create and activate a virtual environment:
     ```bash
     python -m venv venv
     source venv/bin/activate   # Linux/Mac
     venv\Scripts\activate      # Windows
     ```
   - Install dependencies:
     ```bash
     pip install -r requirements.txt
     ```

## Coding Standards
- Follow [PEP 8](https://peps.python.org/pep-0008/) style guidelines.
- Run `flake8` for linting before submitting.
- Write unit tests using `pytest`.
- Ensure all CI/CD checks pass before submitting a PR.

## How to Contribute
1. Fork this repository.
2. Pick an issue labeled `good-first-issue`.
3. Write code and tests.
4. Submit a PR with:
   - Clear description of changes.
   - Screenshots (if UI-related).
   - Reference to the issue number.

We welcome contributions of all sizes, from fixing typos to adding new features!
