# System Specification

## Project Title
MangaBookStore – Online Manga Retail System

## Domain
E-Commerce / Online Retail (Anime and Manga Industry)

## Domain Description
The manga retail domain involves businesses that sell Japanese comic books (manga) to customers. Manga has become globally popular due to the rise of anime culture. Customers often purchase manga volumes online to build collections or read their favorite series.

An online manga bookstore provides users with a convenient way to browse manga titles, view descriptions, check availability, and purchase books from anywhere. The system must manage product listings, customer accounts, orders, and inventory.

This system aims to simulate a digital marketplace where manga books can be browsed and purchased.

## Problem Statement
Many anime and manga fans struggle to find centralized platforms dedicated specifically to manga purchases. Existing online marketplaces may not specialize in manga, making it difficult to discover series, track volumes, or find niche titles.

The MangaBookStore system aims to provide a dedicated platform for manga enthusiasts where they can easily browse manga titles, add books to a shopping cart, and place orders through an organized and user-friendly system.

The system will also allow administrators to manage inventory, update manga listings, and monitor customer orders.

## Individual Scope
This project will implement a simplified version of an online manga store including the following features:

User Features:
- User registration and login
- Browse manga catalog
- Search manga titles
- View manga details
- Add manga to shopping cart
- Place orders

Admin Features:
- Add new manga titles
- Update manga information
- Remove manga listings
- View and manage customer orders

The system will be developed as a web-based application demonstrating the use of modular architecture, database management, and system design principles.

Due to the individual scope of this project, advanced features such as real payment gateways and shipping integration will be simulated rather than fully implemented.
