# System Architecture

## Project Title
MangaBookStore – Online Manga Retail System

## Domain
E-Commerce / Online Retail (Anime and Manga Industry)

## Problem Statement
The MangaBookStore system aims to provide a centralized platform where manga fans can browse, search, and purchase manga volumes online. The system will manage product listings, user accounts, shopping carts, and order processing.

## Individual Scope
The system will implement the core functionality required for a basic online manga store including catalog browsing, shopping cart functionality, and order management.
